Our Library
Southwest Petroleum University (SWPU) Library consists of Nanchong Campus Library and Chengdu Campus Library. Chengdu Campus Library comprises the main library and a branch library. The main library, whose construction started in September 2002, was put into use in February 2008, while the branch library was put into use in December 2016.



SWPU Library features oil and gas literature, though it has a collection which covers a wide range of disciplines, including science, engineering, management, economics, liberal arts, law and education. Currently, there are 42 professional databases, and 1.9 million volumes of printed books and journals, not including those (about 170,000) acquired by the individual schools on the two campuses.



This library provides a series of services, such as online bibliographic retrieval, remote access, self-help borrowing and returning, document delivery, interlibrary loan, information consulting, sci-tech novelty retrieval, resources recommendation, and so on. Besides, the library holds regular training workshops for different groups of readers, such as for freshmen on how to use the library, and for seniors on how to take full advantage of the library resources to carry out graduation projects or write graduation theses.



Meanwhile, the library initiates annual activities around the "4.23 World Reading Day" to promote the use of the abundant resources. Moreover, it provides information services for local scientific research institutions.
